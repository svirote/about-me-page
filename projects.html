Weather app

This weather app allows users to access weather data for over 200,000 cities, by typing the city name or using the geolocation. The results are avaliable in either metric or imperial units. It is also possible to chose the language to be used (english, french or portuguese), and to select 3 favorite cities and see the current weather of each at the same time. The 5 day forecast is avaliable one city ate the time.
